\# 🚗 Autolavado App



Autolavado App is a cross-platform application developed with .NET MAUI to simplify the management of a car wash business. The application provides tools for organizing services, managing customers, and improving daily operations through an intuitive interface.



\## ✨ Features



\- 🚗 Vehicle registration

\- 👤 Customer management

\- 🧾 Service management

\- 📅 Service history

\- 💲 Price management

\- 📱 Cross-platform interface



\## 🛠️ Technologies



\- .NET MAUI

\- C#

\- XAML

\- MVVM Architecture

\## 🚀 Getting Started



\### Requirements



\- Visual Studio 2022

\- .NET 8 SDK

\- .NET MAUI Workload



\### Installation



Clone the repository:



```bash

git clone https://github.com/jimenasdmg/autolavado-app.git

```



Open the solution in Visual Studio.



Restore NuGet packages.



Run the application on Windows or Android.



\## 🎯 Purpose



This project was developed as an academic application to practice cross-platform mobile development using .NET MAUI and the MVVM architectural pattern.
